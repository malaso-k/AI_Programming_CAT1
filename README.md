## Featured Scripts

### Student Information System
- **Description:** A simple program that collects and organizes student details.  
- **Highlights:**
  - Prompts the user for input in a friendly way.
  - Stores information in a structured format.
  - Allows updates and retrieval of records.
- **Takeaway:** Shows how programming can be used to manage everyday tasks like keeping track of student data.

### Area of a Circle
- **Description:** A short script that calculates the area of a circle when given a radius.  
- **Highlights:**
  - Applies the formula πr² directly in code.
  - Encourages user interaction through input prompts.
  - Demonstrates how math and programming connect.
- **Takeaway:** Reinforces the idea that even small scripts can solve practical problems.
